# FlexiOffice - Smart Hybrid Work Platform

FlexiOffice is a modern workspace management platform designed to help teams collaborate intentionally in a hybrid world.

## Features
- **Live Floorplan**: Real-time map of office desks, meeting rooms, and parking.
- **AI Workspace Advisor**: Personalized WFH/Office recommendations based on collaboration needs.
- **Team Presence**: Coordinate anchor days and see where your colleagues are working.
- **Rewards Hub**: Incentivizing sustainable choices like carpooling and WFH streaks.

## Setup & Documentation
- [Integration Setup Guide](docs/INTEGRATIONS.md): How to connect Calendar, Slack, HRIS, and Building Access.
- **Admin Access**: To access the Floorplan Editor, navigate to **Profile Settings** and toggle **Admin Status**.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Firestore
- **Auth**: Firebase Authentication
- **AI**: Genkit (Gemini 2.5 Flash)
- **UI**: ShadCN UI + Tailwind CSS
